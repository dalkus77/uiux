| 화면 ID | 화면명 | 파일명 | 구현 상태 | 비고 |

## 실행 방법

### 설치 및 실행

```bash
# 프로토타입 폴더로 이동
cd prototype

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 `http://localhost:5173` 접속
