# 1단계: 업무분석 및 요구사항 정의

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 사업명 | AI기반 건강습관 개선정보 제공 솔루션 |
| 목적 | AI 기술을 활용하여 사용자의 건강습관을 분석하고 개선 정보를 제공하는 서비스 구축 |
| 범위 | 건강 데이터 수집, AI 기반 분석, 맞춤형 건강 정보 제공, 습관 추적 및 피드백 |
| 대상 사용자 | 건강 관심이 높은 일반인, 만성질환 관리자, 건강증진 프로그램 참여자 |
| 주요 기능 | 건강 데이터 입력/수집, AI 분석, 맞춤형 추천, 습관 추적, 리포트 생성 |

## 업무 현황 분석

| 구분 | 현재 상태 | 문제점 | 개선 방향 |
|------|----------|--------|----------|
| 건강 정보 수집 | 수동 입력, 분산된 데이터 | 데이터 입력 부담, 일관성 부족 | 자동 수집 및 통합 관리 |
| 건강 상태 분석 | 주관적 판단, 전문가 상담 필요 | 접근성 낮음, 비용 부담 | AI 기반 자동 분석 |
| 개선 방안 제시 | 일반화된 정보 제공 | 개인 맞춤 부족, 실행 어려움 | 개인화된 추천 및 실행 가이드 |
| 습관 추적 | 수동 기록, 지속성 낮음 | 기록 누락, 동기 부족 | 자동 추적 및 동기부여 |

## 요구사항 정의서

### 기능 요구사항 (FR)

| ID | 요구사항명 | 상세설명 | 우선순위 | 비고 |
|----|-----------|----------|----------|------|
| FR-001 | 건강 데이터 입력 | 사용자 건강 정보(신체지수, 운동, 식단 등) 입력 | 필수 | 다양한 입력 방식 지원 |
| FR-002 | 건강 데이터 자동 수집 | 웨어러블 기기, 앱 연동을 통한 자동 데이터 수집 | 권장 | API 연동 |
| FR-003 | AI 건강 분석 | 입력된 데이터를 기반으로 건강 상태 분석 | 필수 | AI 모델 활용 |
| FR-004 | 맞춤형 건강 정보 추천 | 개인 건강 상태에 맞는 정보 및 개선 방안 제시 | 필수 | AI 추천 엔진 |
| FR-005 | 습관 추적 | 건강 습관(운동, 식단, 수면 등) 추적 및 기록 | 필수 | 시각화 제공 |
| FR-006 | 목표 설정 및 관리 | 건강 목표 설정 및 달성도 추적 | 필수 | 진행률 표시 |
| FR-007 | 리포트 생성 | 건강 상태 리포트 및 트렌드 분석 | 권장 | 시각화 차트 |
| FR-008 | 알림 및 리마인더 | 건강 습관 실천을 위한 알림 기능 | 권장 | 개인화된 알림 |
| FR-009 | 커뮤니티 기능 | 사용자 간 정보 공유 및 소통 | 선택 | 향후 확장 |
| FR-010 | 전문가 상담 연결 | 필요시 전문가 상담 서비스 연결 | 선택 | 외부 서비스 연동 |
