이 `prototype` 폴더는 이전 React 프로토타입입니다.

## 현재 사용하는 폴더

- ✅ **`app/`** - 순수 HTML/CSS/JS 프론트엔드 (현재 사용)
- ✅ **`backend/`** - Python Flask 백엔드 (현재 사용)

## 이 폴더의 파일들

이 폴더의 파일들은 참고용으로만 남겨두었습니다.
실제 프로젝트 실행에는 사용하지 않습니다.

## 프로젝트 실행 방법

프로젝트 루트 폴더의 `start.bat`를 실행하세요:

```
프로젝트/
└── start.bat  ← 이 파일을 실행하세요!
```

이 파일은 Python만 사용하며 Node.js가 필요 없습니다.

- 이전 React 프로토타입은 Node.js 18 이상이 필요했습니다
