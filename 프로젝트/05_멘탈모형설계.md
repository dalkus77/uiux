# 5단계: 멘탈모형 설계

## 멘탈모형 다이어그램

**서비스명:** AI기반 건강습관 개선정보 제공 솔루션에 대한 멘탈모형

### 3계층 구조표

| 과업 단계 | 사용자 니즈 | 시스템 기능 |
|----------|-----------|------------|
| **1. 내 건강 상태 파악하기** | | |
| | 내가 지금 건강한지 확인하고 싶어 | 건강 상태 대시보드 표시 |
| | 어떤 부분이 문제인지 알고 싶어 | 건강 지표 분석 및 시각화 |
| | 예전과 비교해서 어떻게 변했는지 보고 싶어 | 건강 트렌드 차트 제공 |
| **2. 건강 정보 알아보기** | | |
| | 내 상황에 맞는 건강 정보를 알고 싶어 | 맞춤형 건강 정보 추천 [AI] |
| | 왜 이 정보가 나에게 좋은지 이해하고 싶어 | 추천 근거 설명 제공 [AI] |
| | 실행 가능한 방법을 알고 싶어 | 단계별 실행 가이드 제공 |
| **3. 건강 습관 실천하기** | | |
| | 오늘 무엇을 해야 할지 알고 싶어 | 일일 건강 목표 제시 |
| | 내가 잘하고 있는지 확인하고 싶어 | 습관 실천 현황 표시 |
| | 계속할 수 있도록 격려받고 싶어 | 동기부여 메시지 및 알림 |
| **4. 건강 개선 확인하기** | | |
| | 내 노력이 효과가 있는지 확인하고 싶어 | 건강 개선 리포트 생성 |
| | 얼마나 개선되었는지 보고 싶어 | 전후 비교 차트 제공 |
| | 다음 목표를 설정하고 싶어 | 목표 설정 및 관리 기능 |
| **5. 지속적으로 관리하기** | | |
| | 건강 정보를 쉽게 기록하고 싶어 | 간편 데이터 입력, 자동 수집 |
| | 중요한 것을 놓치지 않고 싶어 | 알림 및 리마인더 기능 |
| | 전문가의 도움이 필요할 때 연결받고 싶어 | 전문가 상담 연결 기능 |

## 사용자 용어 vs 시스템 용어 대응표

| 사용자 용어 (일상어) | 시스템 용어 (전문어) | 적용 위치 |
|-------------------|-------------------|----------|
| 내 건강 상태 파악하기 | 건강 상태 분석 | 메뉴명 |
| 건강 정보 알아보기 | 건강 정보 추천 | 메뉴명 |
| 건강 습관 실천하기 | 습관 추적 및 관리 | 메뉴명 |
| 건강 개선 확인하기 | 리포트 및 분석 | 메뉴명 |
| 지속적으로 관리하기 | 데이터 관리 및 설정 | 메뉴명 |
| 내가 지금 건강한지 확인 | 건강 상태 대시보드 | 화면 제목 |
| 내 상황에 맞는 정보 | 맞춤형 추천 | 버튼/카드 제목 |
| 왜 좋은지 설명 | 추천 근거 | 설명 영역 |
| 오늘 무엇을 해야 할지 | 일일 목표 | 카드/위젯 |
| 잘하고 있는지 확인 | 실천 현황 | 진행률 표시 |
| 노력이 효과가 있는지 | 건강 개선 분석 | 리포트 제목 |
| 얼마나 개선되었는지 | 전후 비교 | 차트 제목 |
| 쉽게 기록하기 | 간편 입력 | 버튼명 |
| 놓치지 않기 | 알림 설정 | 설정 메뉴 |

## IA 설계 가이드라인

### 메뉴 구조 제안

| 과업 단계 | 제안 메뉴명 | 포함 기능 |
|----------|-----------|----------|
| 과업단계 1 | 내 건강 상태 | 건강 대시보드, 건강 지표 분석, 트렌드 확인 |
| 과업단계 2 | 건강 정보 알아보기 | 맞춤형 추천, 추천 근거, 실행 가이드 |
| 과업단계 3 | 건강 습관 실천하기 | 일일 목표, 실천 현황, 동기부여 |
| 과업단계 4 | 건강 개선 확인하기 | 개선 리포트, 전후 비교, 목표 설정 |
| 과업단계 5 | 관리하기 | 데이터 입력, 알림 설정, 전문가 연결 |

### 정보 우선순위

| 순위 | 정보/기능 | 이유 |
|------|----------|------|
| 1 | 현재 건강 상태 요약 | 사용자가 가장 먼저 궁금해하는 것 |
| 2 | 오늘 해야 할 일 | 즉시 실행 가능한 액션 |
| 3 | 건강 개선 진행 상황 | 지속적 동기부여 |
| 4 | 맞춤형 건강 정보 추천 | 개인화된 가치 제공 |
| 5 | 상세 분석 및 리포트 | 깊이 있는 정보 필요 시 |

### 레이블링 원칙

| 원칙 | 설명 | 예시 |
|------|------|------|
| 1. 사용자 언어 사용 | 전문용어 대신 일상어 | "건강 상태 파악" vs "건강 분석" |
| 2. 행동 중심 표현 | 명사보다 동사형 | "알아보기", "실천하기", "확인하기" |
| 3. 간결성 | 3단어 이내 | "내 건강 상태", "오늘 할 일" |
| 4. 긍정적 표현 | 부정보다 긍정 | "건강 개선" vs "건강 문제 해결" |
| 5. 구체성 | 추상적 표현 지양 | "오늘 운동하기" vs "건강 관리" |
