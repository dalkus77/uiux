# Node.js 12.14.0 제거 가이드

## 자동 제거 (권장)

프로젝트 폴더에 있는 `remove_nodejs.bat` 파일을 실행하세요.

**주의**: 관리자 권한으로 실행하는 것을 권장합니다.

1. `remove_nodejs.bat` 파일을 마우스 우클릭
2. "관리자 권한으로 실행" 선택
3. 화면의 안내에 따라 진행

## 수동 제거 방법

### 방법 1: 폴더 직접 삭제

1. `C:\node-v12.14.0-win-x64` 폴더 삭제
   - 파일 탐색기에서 해당 폴더 찾기
   - 삭제 (관리자 권한 필요할 수 있음)

### 방법 2: PATH 환경 변수에서 제거

1. **환경 변수 편집**
   - Windows 키 + R
   - `sysdm.cpl` 입력 후 Enter
   - "고급" 탭 → "환경 변수" 클릭

2. **사용자 변수 편집**
   - "Path" 변수 선택 → "편집" 클릭
   - `C:\node-v12.14.0-win-x64` 항목 찾기
   - 선택 후 "삭제" 클릭
   - "확인" 클릭

3. **시스템 변수 확인** (선택사항)
   - 시스템 변수의 Path에도 Node.js 경로가 있는지 확인
   - 있다면 동일하게 제거

### 방법 3: 제어판에서 제거

1. **제어판 열기**
   - Windows 설정 → 앱 → 앱 및 기능
   - 또는 제어판 → 프로그램 제거

2. **Node.js 찾기**
   - 검색창에 "node" 입력
   - Node.js 관련 프로그램 찾기

3. **제거**
   - Node.js 선택 → 제거 클릭

## 제거 확인

새 명령 프롬프트를 열고 다음 명령 실행:

```bash
node --version
```

다음과 같은 메시지가 나오면 제거 완료:
```
'node'은(는) 내부 또는 외부 명령, 실행할 수 있는 프로그램, 또는 배치 파일이 아닙니다.
```

## 중요 사항

### ⚠️ 현재 프로젝트는 Node.js가 필요 없습니다

이 프로젝트는:
- ✅ 순수 HTML/CSS/JavaScript 사용
- ✅ Python Flask 백엔드 사용
- ❌ Node.js 불필요

따라서 Node.js를 제거해도 프로젝트 실행에 문제가 없습니다.

### 🔄 PATH 변경사항 적용

PATH 환경 변수를 변경한 경우:
- 새 명령 프롬프트를 열어야 변경사항이 적용됩니다
- 또는 컴퓨터 재시작

### 📝 참고

- Node.js를 완전히 제거하지 않아도 프로젝트는 정상 작동합니다
- 다른 프로젝트에서 Node.js가 필요할 수 있으므로, 필요시 다시 설치할 수 있습니다
