## 사용자 플로우 (핵심 시나리오)

### 플로우 1: 건강 상태 확인 및 맞춤형 추천 받기

**목표:** 김민수의 목표 - 건강 상태를 파악하고 개선 방법을 찾기

```
[시작: 앱 실행]
   │
   ▼
┌─────────────┐
│  메인 화면   │
│ (대시보드)  │
└──────┬──────┘
       │
       ▼
   <데이터 있음?>
       │ Yes        │ No
       ▼            ▼
┌─────────────┐ ┌─────────────┐
│ 건강 상태    │ │ 데이터 입력  │
│ 분석 결과    │ │ 화면         │
│   [AI]      │ └──────┬──────┘
└──────┬──────┘        │
       │               │
       └───────┬───────┘
               ▼
        ┌─────────────┐
        │ 맞춤형 추천  │
        │   [AI]      │
        └──────┬──────┘
               │
               ▼
        ┌─────────────┐
        │ 추천 근거    │
        │ 확인 화면    │
        └──────┬──────┘
               │
               ▼
        ┌─────────────┐
        │ 실행 가이드  │
        │ 확인 완료    │
        └─────────────┘
```

**단계별 설명:**

| 단계 | 화면/기능 | 사용자 행동 | 시스템 반응 | AI 개입 |
|------|----------|------------|------------|---------|
| 1 | 메인 화면 (대시보드) | 앱 실행, 건강 상태 확인 | 건강 지표 요약 표시 | O |
| 2 | 건강 상태 분석 | 상세 분석 결과 확인 | 건강 지표 분석, 이상 징후 표시 | O |
| 3 | 맞춤형 추천 | 추천 정보 확인 | 개인화된 건강 정보 추천 | O |
| 4 | 추천 근거 확인 | 추천 이유 확인 | 추천 근거 및 설명 제공 | O |
| 5 | 실행 가이드 | 실행 방법 확인 | 단계별 실행 가이드 제공 | |

### 플로우 2: 건강 습관 실천 및 추적

**목표:** 일일 건강 목표를 실천하고 기록하기

```

   │
   ▼
┌─────────────┐
│  오늘 할 일  │
│  목표 화면  │
└──────┬──────┘
       │
       ▼
   <목표 실천?>
       │ Yes        │ No
       ▼            ▼
┌─────────────┐ ┌─────────────┐
│ 습관 기록    │ │ 알림 설정    │
│ 입력 화면    │ │ (리마인더)   │
└──────┬──────┘ └──────┬──────┘
       │               │
       └───────┬───────┘
               ▼
        ┌─────────────┐
        │ 실천 현황    │
        │ 진행률 표시  │
        └──────┬──────┘
               │
               ▼
        ┌─────────────┐
        │ 동기부여     │
        │ 메시지 표시  │
        └─────────────┘
```

## 내비게이션 구조

### GNB (Global Navigation Bar)

| 순서 | 메뉴명 (사용자 언어) | 아이콘 | 비고 |
|------|-------------------|--------|------|
| 1 | 내 건강 상태 | 🏥 | 메인 화면 |
| 2 | 건강 정보 알아보기 | 💡 | AI 추천 |
| 3 | 건강 습관 실천하기 | ✅ | 일일 목표 |
| 4 | 건강 개선 확인하기 | 📊 | 리포트 |
| 5 | 관리하기 | ⚙️ | 설정 |

### 컨텍스트 내비게이션

| 상황 | 표시 요소 | 액션 |
|------|----------|------|
| 건강 상태 이상 시 | 경고 배지, 상세 분석 버튼 | 건강 지표 분석 화면 이동 |
| 추천 정보 확인 시 | "왜 추천되었나요?" 버튼 | 추천 근거 화면 이동 |
| 목표 달성 시 | 성취 배지, 다음 목표 제시 | 목표 설정 화면 이동 |

## IA-Pain Point 매핑

| Pain Point | 해결 메뉴/기능 | 해결 방식 |
|-----------|---------------|----------|
| PP-001 (데이터 입력 어려움) | 관리하기 > 데이터 입력, 자동 수집 설정 | 자동 데이터 수집, 간편 입력 폼 |
| PP-002 (AI 결과 이해 어려움) | 건강 정보 알아보기 > 추천 근거 보기 | 설명가능한 AI, 사용자 언어 설명 |
| PP-003 (추천 비현실적) | 건강 습관 실천하기 > 오늘 할 일, 실행 가이드 | 작은 목표, 실행 가능성 검증 |
| PP-004 (지속적 관리 어려움) | 건강 습관 실천하기 > 실천 현황, 건강 개선 확인하기 | 게이미피케이션, 진행률 시각화 |
| PP-005 (가입 복잡) | 관리하기 > 간편 설정 | 최소 설정, 원클릭 가입 |
| PP-006 (신뢰성 불확실) | 건강 정보 알아보기 > 추천 근거 보기 | 의료진 인증, 근거 제시 |
