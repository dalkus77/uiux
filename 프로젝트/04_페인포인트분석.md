## 우선순위 매트릭스

```
        빈도 높음
            │
     ┌──────┼──────┐
     │ PP-004│ PP-001│
     │ 중요  │ 긴급+중요│
심각도├──────┼──────┤심각도
 낮음 │ PP-008│ PP-002│ 높음
     │ 관찰  │ 주시   │
     └──────┼──────┘
            │
        빈도 낮음
```

| 우선순위 | Pain Point ID | 조치 방향 |
|---------|--------------|----------|
| 1순위 (긴급) | PP-001, PP-002 | 즉시 해결 - 자동 데이터 수집, AI 설명가능성 강화 |
| 2순위 (중요) | PP-003, PP-004 | 계획적 해결 - 실행 가능한 목표, 게이미피케이션 |
| 3순위 (주시) | PP-005, PP-006 | 모니터링 후 대응 - 간소화된 가입, 신뢰성 표시 |
| 4순위 (관찰) | PP-007, PP-008 | 장기 검토 - 보안 강화, 정보 필터링 |

## Pain Point 상세 분석 (상위 3개)

### PP-001: 데이터 입력 항목이 많고 반복적임

| 항목 | 내용 |
|------|------|
| Pain Point | 건강 정보 입력 시 항목이 많고 매번 반복 입력해야 함 |
| 발생 단계 | 4단계 (데이터 입력) |
| 영향받는 페르소나 | P1 (김민수), P2 (이영희) - 공통 |
| **근본 원인** | 수동 입력 방식, 데이터 자동 수집 기능 부재, 입력 폼 설계 미흡 |
| 현재 대안 | 복사/붙여넣기, 기본값 저장, 입력 히스토리 활용 |
| **AI 해결 가능성** | 높음 |
| AI 해결 방향 | 웨어러블 기기 자동 연동, 스마트폰 센서 데이터 활용, 이전 입력 패턴 학습하여 자동 완성, 음성 입력 지원 |
| 기대 효과 | 입력 시간 80% 감소, 사용자 만족도 향상, 데이터 정확도 향상 |

### PP-002: AI 분석 결과의 이해와 신뢰가 어려움

| 항목 | 내용 |
|------|------|
| Pain Point | AI 분석 결과가 어떻게 나왔는지 이해하기 어렵고, 신뢰할 수 있는지 불확실함 |
| 발생 단계 | 5단계 (AI 분석 확인) |
| 영향받는 페르소나 | P1 (김민수), P2 (이영희) - 공통 |
| **근본 원인** | AI 결과에 대한 설명 부족, 전문 용어 사용, 신뢰도 지표 부재, 근거 제시 미흡 |
| 현재 대안 | 결과만 표시, 간단한 설명 텍스트, FAQ 제공 |
| **AI 해결 가능성** | 높음 |
| AI 해결 방향 | 설명가능한 AI(XAI) 적용, 결과 도출 근거 시각화, 신뢰도 점수 표시, 사용자 언어로 설명 제공, 단계별 상세 설명 제공 |
| 기대 효과 | 사용자 신뢰도 향상, 서비스 이해도 향상, 의료진과의 소통 개선 |

### PP-003: 추천/계획이 비현실적이거나 실행 어려움

| 항목 | 내용 |
|------|------|
| Pain Point | AI가 추천한 건강 개선 방법이 현실적으로 실행하기 어렵거나 비현실적임 |
| 발생 단계 | 6단계 (추천 실행) |
| 영향받는 페르소나 | P1 (김민수), P2 (이영희) - 공통 |
| **근본 원인** | 사용자 라이프스타일 고려 부족, 일반화된 추천, 실행 가능성 검증 부재 |
| 현재 대안 | 사용자가 직접 선택, 추천 무시 |
| **AI 해결 가능성** | 중간 |
| AI 해결 방향 | 사용자 라이프스타일 데이터 학습, 작은 단계별 목표 제시, 실행 가능성 점수 제공, 사용자 피드백 반영하여 추천 개선 |
| 기대 효과 | 추천 실행률 향상, 사용자 만족도 향상, 건강 개선 효과 증대 |

## HMW (How Might We) 질문

| Pain Point | HMW 질문 | 해결 방향 |
|-----------|----------|----------|
| PP-001 | 어떻게 하면 사용자가 최소한의 입력만으로 건강 데이터를 수집할 수 있을까? | 자동 데이터 수집, 스마트 기본값, 패턴 학습 |
| PP-002 | 어떻게 하면 사용자가 AI 분석 결과를 쉽게 이해하고 신뢰할 수 있을까? | 설명가능한 AI, 시각화, 사용자 언어 설명 |
| PP-003 | 어떻게 하면 사용자가 실제로 실행 가능한 건강 개선 방법을 추천받을 수 있을까? | 라이프스타일 기반 추천, 작은 목표, 실행 가능성 검증 |
| PP-004 | 어떻게 하면 사용자가 지속적으로 건강 관리를 할 수 있도록 동기를 부여할 수 있을까? | 게이미피케이션, 진행 상황 시각화, 성취감 제공 |
| PP-005 | 어떻게 하면 사용자가 빠르고 간단하게 서비스를 시작할 수 있을까? | 원클릭 가입, 소셜 로그인, 최소 설정 |
