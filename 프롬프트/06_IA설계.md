# 6단계: IA(정보구조) 설계

## 단계 목표
멘탈모형과 Pain Point를 기반으로 서비스의 정보 구조를 설계하여 "어떻게 구성할 것인가"를 정의합니다.

---

## 입력 필요 산출물
- 1단계: 요구사항 정의서 (기능 요구사항)
- 4단계: Pain Point 분석 (우선순위, HMW 질문)
- 5단계: 멘탈모형 설계 (카테고리 구조, 사용자 용어, 가이드라인)

---

## 프롬프트

### 6-1. IA 설계 프롬프트

```
AI서비스 UIUX 설계 중. 6단계 정보구조(IA) 설계.

[멘탈모형]
{5단계 멘탈모형 다이어그램 및 가이드라인 붙여넣기}

[요구사항 정의서 - 기능 요구사항]
{1단계 기능 요구사항 붙여넣기}

[Pain Point 우선순위]
{4단계 우선순위 목록 붙여넣기}

멘탈모형 기반으로 IA 설계:

1. 메인 메뉴 구조 (1depth) - 사용자 언어 사용
2. 하위 메뉴 구조 (2depth)
3. 각 메뉴별 주요 기능
4. 메뉴 간 연관관계

트리 구조로 표현. AI 기능은 [AI] 표시.
사용자 관점의 레이블 사용.
```

### 6-2. 사용자 플로우 프롬프트

```
IA 기반으로 핵심 사용자 플로우 작성:

시나리오: {페르소나가 목표 달성하는 과정}

플로우차트 형식으로:
[시작] → [화면/기능] → [분기점] → [화면/기능] → [완료]

AI 개입 지점 명시.
```

### 6-3. 내비게이션 설계 프롬프트 (선택)

```
IA 기반으로 내비게이션 설계:
1. GNB(Global Navigation Bar) 구성
2. LNB(Local Navigation Bar) 구성
3. 컨텍스트 내비게이션
4. 검색/필터 구조

페르소나 기술 숙련도 고려하여 복잡도 조절.
멘탈모형의 사용자 용어를 레이블에 적용.
```
